# Most Commone Primitive Types

### int

An integer (whole number) between -2^31 and +2^31 - 1.

Smallest value of int: -2,147,493,648 

Largest value of in: 2,147,493,647

It takes up 32 bits.

`int i = 122;`

### boolean
